    锁

    锁名称 思想 应用场景
    悲观锁 悲观锁不允许在获得资 源操作机会后被其他线程抢走机会
    乐观锁 乐观锁允许在获得资源机会前被其他线程抢走机会
    • 悲观锁本质是通过信号量机制实现线程对资源的独占性
    • 乐观锁本质是通过比较放弃实现多个线程对资源操作的连续性
